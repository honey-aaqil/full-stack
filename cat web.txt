13.<form></form>:- it used to get info,data
    attribute:-
             1. action: Specifies the URL where the form data will be sent for processing.
            
             2. method: Defines the HTTP method used when submitting the form (usually either "get" or "post").
            
             3. name: Assigns a name to the form, which can be used for identifying the form in scripts or styling.
            
             4. target: Specifies where to display the response received after submitting the form (e.g., "_self", "_blank", "_parent", "_top").
            
             5. enctype: Defines the content type used to encode the form data before sending it to the server (e.g., "application/x-www-form-urlencoded", "multipart/form-data", "text/plain").
            
             6. autocomplete: Specifies whether the browser should automatically complete form fields based on previous input.
            
             7. novalidate: Indicates that the form should not be validated by the browser before submission.
            
             8. onsubmit: Specifies a script to be executed when the form is submitted.
            
             9. onreset: Specifies a script to be executed when the form is reset.
            
             10. accept-charset: Specifies the character encodings that are to be used for form submission.
            
             11. accesskey: Defines a keyboard shortcut to activate or focus on the form element.
            
             12. class: Assigns one or more classes to the form for styling purposes.
            
             13. id: Assigns a unique identifier to the form, which can be used for targeting with scripts or styling.
            
             14. style: Defines inline CSS styles for the form.
            
             15. title: Provides additional information about the form (often displayed as a tooltip).
            
             16. dir: Specifies the direction of the form's text (ltr or rtl).
            
             17. lang: Specifies the language of the form's content.
            
             18.tabindex: Specifies the tab order of the form element.
            
             19.role: Defines the role of the form (useful for accessibility)
